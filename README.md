# hifile

#### 一 、介绍
hifile是一款基于Linux的开源文件防篡改软件，对目录文件保护实时生效，而且配置简单。

#### 二 、主要用途
##### 1、对重要目录只读  

```
可以对/etc/、网站等敏感目录进行监控，设置只能读，不能修改和删除。
```


##### 2、防止黑客利用未知RCE漏洞，落地木马或者修改服务器重要文件

    现在高级漏洞都不直接公开了，上升为国家之间情报战略，秘密潜伏的木马文件比破坏的多得多。



#### 三、系统内核要求
要求内核版本大于5.10，推荐2021年以后的linux发行版本：

Ubuntu 23.04+

Fedora 31+

RHEL 8.2+

Debian 11+...
